Word Count

Marc Davies | GitHub | LinkedIn

Instructions:

Create a website that lets users input a block of text and returns a list of all the unique words and how many times they appeared.

For example, if the user inputs "hello world world", the page should show:

  • hello 1
  • world 2

Extra:

  • Order the list by the number of appearances - greatest to least.
  • For words that appear the same amount of times, order by which word appeared first.
